# Call Scripts — Starter Set for A/B Testing

These are frameworks, not word-for-word scripts. The Phone SDR agent (powered by Retell AI) adapts in real-time using function calling. The weekly self-improvement loop evolves these based on what leads to conversations.

## Opening — Variant A (Trigger-Based)

```
"Hi [Name], this is [Your Name] with [Company].
I saw [specific trigger — email open, LinkedIn post, funding news].
Got 30 seconds?"
```

## Opening — Variant B (Pain Point Lead)

```
"Hi [Name], this is [Your Name] with [Company].
Quick question — are you still handling [pain point] the old way, or have you automated that yet?
```

## If They Say "Go Ahead" / "Sure"

```
"Great. In 30 seconds:

We help [industry] companies [specific outcome].
[Similar company] went from [before state] to [after state] in [timeframe].

Based on what I know about [Company], I think there might be a fit.
Would it make sense to do a quick 15-min call this week to see?"
```

## If They Say "I'm Busy"

```
"Totally get it — when's a better 2 minutes? I can call back."
[If they give a time, log it and call back. If they say "never," thank them and hang up.]
```

## Common Objections

### "We already have a solution"
```
"That's great — most companies I talk to do.
Quick question: is it handling [specific gap your solution addresses]?
If yes, no need to chat. If not, might be worth a quick look."
```

### "Send me an email"
```
"Happy to — I actually sent one [yesterday/earlier this week].
Subject line was '[subject]'. I'll re-send it right now.
If it looks interesting, want to grab 15 minutes next week?"
```

### "Not interested"
```
"Fair enough. Mind if I ask — is it the timing, or is [pain point] genuinely not a priority right now?"
[Listen. Sometimes "not interested" means "not now" and you can set a follow-up.]
```

### "How much does it cost?"
```
"Depends on the scope, but most clients are in the [$X-$Y] range.
The better question is what it costs you NOT to fix [pain point] —
[similar company] was losing [$X/month] before we helped them.
Want to do a quick call to see if the math works for you?"
```

## Speed-to-Lead Script (When Calling Within 5 Min of Email Open)

```
"Hi [Name], this is [Your Name] with [Company].
I just sent you an email about [topic] — figured it's easier to explain in 30 seconds than make you read.
Got a quick minute?"
```

**Why this works**: They literally just opened your email. They know who you are. The connection rate is 10x higher.

---

## A/B Testing Notes

- Test openers first (biggest impact on whether they stay on the line)
- Test closes second (biggest impact on whether they book)
- Need 20+ calls per variant (calls are lower volume than email)
- Winner = higher meeting-booked rate, not higher connect rate
